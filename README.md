# Student Management Bot

Бот для учёта студентов в Telegram.

## Что делает

- Учёт студентов (ФИО, ПК, расписание)
- Личный кабинет студента
- Админ панель
- Расписание посещений

## Как запустить

1. Склонировать репозиторий
2. Установить зависимости:
```
pip install -r requirements.txt
```
3. Создать файл .env и написать токен от бота
4. Запустить:
```
python main.py
```

## Как получить токен

1. Написать @BotFather в Telegram
2. Создать бота командой /newbot
3. Скопировать токен
4. Узнать свой ID через @userinfobot
5. Вставить всё в .env

## Структура

- main.py - главный файл, запуск бота
- config.py - настройки (токен, админ ID)
- database.py - база данных
- handlers/ - обработчики команд
